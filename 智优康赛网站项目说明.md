## 🎨 设计规范（完整版）

### 1. 按钮规范
**Ghost Cut 风格（通用按钮）：**
```css
background: rgba(255, 255, 255, 0.06);
border: 1px solid rgba(255, 255, 255, 0.15);
border-radius: 4px;  /* 微圆角，禁止胶囊按钮 */
```

**特殊按钮 [预约专家诊断]：**
```css
background: linear-gradient(135deg, #e0e0e0, #a0a0a0);  /* 银白纯色 */
color: #0a0a0a;  /* 深炭色文字 */
```

### 2. 排版规范
- **Section Eyebrow（板块标签）：** 微光胶囊标签(Pill Badge) + 8px 蓝色呼吸圆点
- **禁止：** Section Eyebrow 横线伪元素
- **标题字体大小：** 超大字号（64px-80px Hero 主标题）
- **正文：** 减少长段落文字，注重视觉呈现

### 3. 交互规范
- **H5 端菜单：** 全屏遮罩淡入（Fade-in Overlay），**禁止** SlideDown
- **排名数字：** 不加 # 符号（例如：显示 "1" 而非 "#1"）
- **Logo 区域：** 必须用 `<img>` 占位图 + grayscale + brightness 滤镜，**禁止**纯文本

### 4. 布局规范
- **Hero 首屏：** 必须有 Canvas 动态网络背景（65 节点 + 连线 + 光点流转）
- **客户案例区（PC 端）：** 3 列 grid 全景展开，**禁止**横向滚动
- **服务/技术模块：** 左右分屏布局，position: sticky 配合

## 🏗️ 代码架构说明

### HTML 结构（index.html）

#### 1. Hero 首屏区块（`<section class="hero">`）
- **Canvas 动态背景：** `#hero-canvas` 元素，65 个节点 + 连线动画
- **Logo 滚动条：** `.hero-logos` 容器，7 个 AI 平台 Logo + 名称
- **CTA 按钮：** "联系我们" 按钮，点击弹出模态框收集客户需求

**关键代码片段：**
```html
<!-- Hero CTA 按钮 -->
<button class="btn-ghost" onclick="openModal('contact')">联系我们</button>

<!-- AI 平台 Logo + 名称 -->
<div class="hero-logo-item">
  <img src="images/文心一言.png" alt="文心一言" loading="lazy">
  <span class="logo-name">文心一言</span>
</div>
```

#### 2. 客户案例区块（`<section id="cases">`）
- **PC 端：** 3 列 Grid 布局，全景展开
- **移动端：** 垂直堆叠
- **内容：** 数据增长卡片 + 高管引言

#### 3. 服务体系区块（`<section id="services">`）
- **Sticky Stack Cards：** 粘性层叠卡片效果
- **交互：** 滚动时卡片依次固定

#### 4. 页脚（`<footer>`）
- **AI 平台 Logos：** 与 Hero 区块一致，7 个平台 Logo + 名称
- **公司 Logo：** `images/智优康赛logo.svg`

### CSS 架构（`<style>` 标签内）

#### 关键样式类
```css
/* Hero Logo 项 */
.hero-logo-item {
  display: flex;
  align-items: center;
  gap: 10px;
}

/* Logo 名称文字 */
.logo-name {
  font-size: 13px;
  font-weight: 600;
  color: rgba(255, 255, 255, 0.35);
}

/* 页脚 AI 平台项 */
.ftr-ai-item {
  display: flex;
  align-items: center;
  gap: 8px;
}

/* Marquee 滚动项（客户案例） */
.marquee-item img {
  height: 30px;
  opacity: 0.55;
}
```

#### 响应式断点
- **Desktop: ≥1024px
- **Tablet:** 768px - 1023px
- **Mobile:** ≤ 767px

### JavaScript 功能

#### 1. Canvas 动态网络背景
- **文件：** `$CREATE_OUTPUT_FILE("$SAVE_PATH_TO_YOUR_OUTPUT_FILE_HERE")drawConnections()`
- **功能：** 65 个节点随机运动 + 连线 + 光点流转效果

#### 2. 模态框系统
- **函数：** `openModal(type)` / `closeModal(type)`
- **类型：** 
  - `'contact'` - 联系我们（收集客户需求）
  - `'trial'` - 申请试用
  - `'audit'` - 预约诊断

#### 3. Logo 滚动动画
- **实现：** CSS `@keyframes scroll` + 双倍内容实现无缝滚动
- **速度：** 20-30s 每轮

## ⚠️ 待解决问题

### 1. 手机预览访问（未完成）
**问题：** 用户需要在手机上预览网站效果

**已尝试方案：**
- ❌ 本地服务器（端口 8889）+ 二维码 → 手机无法访问
- ❌ Netlify Drop → 未执行
- ❌ EdgeOne Pages CLI → 安装失败（NODE_OPTIONS 限制）
- ❌ Localtunnel → 失败

**可能原因：**
1. Mac 防火墙阻止了端口访问
2. 手机与 Mac 不在同一 WiFi 网络
3. 路由器隔离了设备间通信

**建议解决方案：**
- 方案 A：使用在线静态托管（Surge.sh / Vercel / GitHub Pages）
- 方案 B：关闭 Mac 防火墙或添加 Python 为例外
- 方案 C：使用 ngrok 或类似内网穿透工具
