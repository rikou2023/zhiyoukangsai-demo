### 视觉特点
- **截图轮播尺寸做大**：占视口 60-70% 宽，让用户看清后台细节
- 用 `aspect-ratio: 16/10`（不是 3/2）以匹配后台仪表盘比例
- 截图边缘加深色容器框 + 浏览器标题栏装饰（fake macOS chrome）
