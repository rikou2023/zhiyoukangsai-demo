### 视觉设计要点
- **证书图与书封面用真实图片，不要做卡通装饰**
- 出版书籍用"立体陈列"效果（轻微 3D 透视，像把书摆在桌面上）
- Google + SEMrush 配图大尺寸展示（占视口 50% 宽度），表达"硬合作"
