智优康赛官网 · 首页 Hero + 背景特效预览
========================================

配色已按您之前指定的「干净电光蓝」方案定稿，本次请您挑选喜欢的背景特效。

如何使用：
1. 双击 index.html，会用浏览器打开（推荐 Chrome / Edge）
2. 页面展示首页 Hero（Logo / 导航 / 主标题 / CTA / AI 平台跑马灯）
3. 右下角有「BG FX」切换栏，共 7 个按钮，挨个点切换不同背景特效
4. 选择会自动记住，刷新页面也保持

7 种背景特效说明：

- ∅ 无              纯主题色背景，最干净
- ◐ 极光            3 个柔光球缓慢漂浮，像极光在背景流动（默认）
- ◉ 神经网络        45 个节点 + 邻近连线，鼠标靠近会激活节点
- ◎ 信号波          Hero 中央向外扩散同心圆，4 圈错时启动
- ◯ 聚光灯          鼠标位置周围有柔光跟随
- ✦ 星尘            220 颗细密小星缓慢漂落 + 左右轻微晃动
- ▦ 网格            隐约网格 + 3 处交点偶尔脉冲发亮

挑完后请告诉我们图标符号或名字（如「我选极光」或「神经网络好」），即可定稿。

技术说明：
- 该文件完全离线，不联网也能打开
- 单文件 272 KB，所有图片 / 样式 / 脚本都已内嵌
- Chrome / Edge / Safari / 手机端都支持
- 鼠标交互特效（神经网络 / 聚光灯）在手机上自动降级
