## 3. 主要问题

### P1 · 移动端导航不可用

CSS 在 768px 以下隐藏了主导航和部分按钮，只显示 hamburger：

- `website/assets/css/index.css:689`

但 JS 目前只切换 hamburger 自身的 `open` 类，没有展开菜单：

- `website/assets/js/index.js:771`

结果是移动端用户看不到 7 项导航，也无法通过顶栏进入各子页。这个属于显示与体验问题。

建议：

- 增加移动端下拉菜单或抽屉菜单。
- hamburger 点击时展开 7 项导航 + 联系我们 + 预约诊断。
- 同步维护 `aria-expanded`。

### P1 · 首页出现了非最终品牌名 `Netconcepts`

客户当前硬约束是公司名使用“智优康赛 / AIconcepts”。首页七大能力标题中出现：

```text
Netconcepts 自研系统
```

位置：`website/index.html:229`

建议改为：

```text
智优康赛自研系统
```

或：

```text
AIconcepts 自研系统
```

### P1 · 合作品牌 Logo 少注入 1 个

文件夹 `website/logo/logo/` 中实际有 78 个 Logo 文件，但 `website/assets/js/index.js` 的 `LOGOS` 数组只列了 77 个。运行时渲染为 154 个 tile，是 77 个 Logo 复制两遍。

缺失文件：

```text
客户合作 logo.png
```

位置：`website/assets/js/index.js:636`

建议：确认该文件是否应该展示；如果客户要求“78 个 Logo 双行反向滚动”，需要补进 `LOGOS` 数组。

### P2 · 顶栏滚动状态使用了玻璃模糊

客户和设计文档都明确不希望玻璃磨砂感。当前滚动后的顶栏使用：

```css
backdrop-filter: blur(16px);
```

位置：`website/assets/css/index.css:25`

这虽然只用于顶栏，不是大面积卡片，但仍有“玻璃磨砂”风险。

建议改为不透明或半透明深色实底，例如：

```css
background: rgba(10, 20, 40, 0.96);
backdrop-filter: none;
```

### P2 · 顶栏下拉指示器尚未实现

设计图里“带下拉指示器的现代化顶栏”是可借鉴细节。目前导航是纯文本，没有下拉箭头。这个不是硬性阻塞，但如果后续子页有二级内容，可以为“GEO 解决方案 / 专属服务”等导航增加小箭头。

### P2 · 表单离线成功逻辑有生产风险

表单提交时，如果 `fetch('/api/contact')` 抛异常，前端会直接展示成功：

位置：`website/assets/js/index.js:748`

这对静态预览友好，但正式上线时可能掩盖后端不可用问题。虽然这次主要验收设计和内容，但建议上线前改成只在明确 demo 模式下才假成功。

## 7. 设计打磨建议

当前不需要重新出设计图，也不建议重做首页整体。页面已经进入“交付前设计打磨”阶段，重点是补齐移动端体验、统一品牌表达、减少客户明确不喜欢的视觉风险，并让系统截图更自然地融入深色页面。

| 优先级 | 设计项 | 建议 |
|---|---|---|
| 高 | 移动端导航设计补齐 | 小屏不能只显示 hamburger 图标，需要展开真实菜单，包含 7 项导航 + 联系我们 + 预约诊断。 |
| 高 | 统一品牌表达 | `Netconcepts 自研系统` 建议改成“智优康赛自研系统”或“AIconcepts 自研系统”，避免客户觉得品牌归属混乱。 |
| 中 | 顶栏滚动状态去玻璃模糊 | 客户明确不喜欢玻璃磨砂，顶栏滚动后建议改为深色实底或高不透明深底。 |
| 中 | 系统演示截图做深色融合 | 系统演示段当前可保留，但浅色截图放在深色页面里略突兀。建议加强深色外框、内阴影、浏览器标题栏和电蓝描边，让截图更像嵌入产品快照。 |
| 中 | 页面节奏再收一点 | 首页内容较多，后半段容易显得资料堆叠。建议统一段落上下留白、标题层级和卡片密度，让每一屏只突出一个重点。 |
| 低 | 顶栏下拉小箭头 | 仅在后续真的有二级菜单时加入，不建议为了装饰硬加。 |
| 低 | Logo 墙补齐 78 个 | 偏内容完整性，但也影响客户要求的“双行 78 Logo”是否兑现。 |

推荐先处理 4 个最能提升客户观感的点：

1. 移动端菜单。
2. `Netconcepts` 品牌字样。
3. 顶栏玻璃模糊。
4. 系统截图与深色页面的融合感。
