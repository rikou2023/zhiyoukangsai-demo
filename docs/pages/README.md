## 各页素材

每个页面目录下都有 `assets/` 子目录，包含该页对应的：
- 客户提供的 docx 文档（**文案权威来源，一字不改**）
- 图片素材（截图 / 证书 / 配图）
- 子目录素材（如出版书籍、系统截图等）

## 重要规则

1. **文案权威**：所有页面文案以 `assets/` 里的 docx 为准，**不允许自己编**
2. **配色锁定**：M · 干净电光蓝（详见全站规范）
3. **表单统一**：所有 CTA 入口（预约诊断/联系我们/需求对接）共用同一表单
4. **背书展示**：证书/出版/Google/SEMrush 这些硬背书集中放在「关于我们」
5. **专属服务流程图**：一模一样照搬，不可重画

## 区别速查

- **解决方案** vs **专属服务**：
  - 解决方案 = 按角色/团队切分（你是哪类团队，我们能为你做什么）
  - 专属服务 = 按交付流程切分（我们的服务机制、流程、4 阶段交付体系）

- **系统演示** vs **专属服务**：
  - 系统演示 = 技术能力 / 看到产品 / 后台截图（让访客感受系统强大）
  - 专属服务 = 服务体系 / 流程 / 交付（让访客理解服务怎么做）

- **客户案例** vs 首页评价段：
  - 首页评价 = 精选呈现（13 条）
  - 客户案例页 = 完整呈现（3 大案例放大 + 全部评价 + 行业根基 + 客户名单）
