## 2. 已敲定的设计决策（客户拍板，不要再问）

### 主标题（最终版）
```
让您的品牌占领 AI [第一]推荐位
       ↑          ↑
   一行整句       "第一" 用电光蓝凸显
```
下面**独立一行**是平台轮换 chip（每 2.8s 切一个）：
```
[Logo Icon] 平台名
```
平台只用国内：Kimi / 豆包 / 文心一言 / DeepSeek / 通义千问 / 元宝 / 智谱清言

### 副标题
```
中国 GEO 第一服务商，让每一个优秀的企业都能被世界看见。
```

### CTA
- **预约诊断**（主，实心电蓝带发光阴影）
- **联系我们**（次，描边）
- 顶栏的"预约诊断"是**白底蓝字**变体（区别于 Hero 主 CTA）
- 顶栏的"联系我们"是深面板背景 + 白字（**不能透明**）

### 配色：Theme M · 干净电光蓝（客户最终拍板，已锁定）
```css
--bg-base:       #0A1428   /* 深海军 */
--bg-surface:    #142036   /* 卡片底 */
--bg-elevated:   #1A2A47
--brand:         #2F66FF   /* 电光蓝 — Logo / 数字 / chip 都用这个 */
--accent:        #2F66FF
--cta:           #2F66FF
--cta-text:      #FFFFFF
--text:          #FFFFFF
--text-muted:    #8B96AD
--border:        #1B2A47
--border-strong: #2D3D62
```
**关键**：
- 整套**只用一种品牌色**（电蓝），不掺红橙绿
- 客户拍板基于一张参考图（电光蓝深海军配色），这是干净科技感的来源
- ⚠️ **客户早期说过"按 Logo 配色来"，但中途看了 13 套备选方案后改主意，明确选了 M（电光蓝），不是 Logo 蓝 #036EB8。最终决策以 M 为准。**
- tokens.css 已锁定为唯一主题，无切换

### 顶栏导航 7 项（客户后期重新拍板）

## 3. 客户多轮反馈历程（避免再走弯路）

按时间顺序：

1. 客户给原始素材包（包括 docx 文案、Logo SVG、PPT、22 张标注截图）
2. 客户说"按 Logo 配色来"+"原网站很丑"+"AI 科技感强" → 提供 13 套配色让客户选
3. 客户**最终选 M · 干净电光蓝**（基于参考图，覆盖早期"按 Logo 来"的说法）
4. 客户对 M 配色 Hero 的批注：
   - 顶栏字"颜色有点浅，要亮白"
   - 顶栏「预约诊断」"不要红色，要白色"
   - 主标题改成 "让您的品牌占领 AI 第一推荐位"（一句），"第一"蓝色凸显
   - 标题下面是 AI Logo 动态
   - 标题字号太大要小
   - Hero 中央 CTA 颜色太浅
5. 客户提供 7 种背景特效让选 → **敲定神经网络**
6. 客户要求：
   - 神经网络节点稍微多一点（45 → 70）
   - 鼠标附近节点跟着鼠标走（不是全部）
   - **只展示国内 AI 平台，砍掉 ChatGPT / Gemini / Claude**

### tryprofound.com 的真实定位（重要修正）
**客户原话不是"参考"，而是"整个网站的设计布局 + 动效可以直接照抄"。**
准确理解：
- ✅ 布局 = 直接抄（顶栏居中导航 / 居中 Hero / 平台名内嵌主标题 / 单列垂直叙事）
- ✅ 动效 = 直接抄（滚动入场 / 平滑过渡 / 节点动画 / 滚动揭示）
- ❌ 配色 = 不抄（用 M 电光蓝，不要 tryprofound 的紫色）
- ❌ 文字 = 不抄（用客户给的中文原文，不要翻译版）
- ⚠️ 客户**明确指定**的地方（如主标题文案/平台 Logo 内嵌/朱砂色按钮等）按客户指定来，不照抄 tryprofound

7. **客户后期发的新需求**（2026-05-19 同步）：
   - **导航重新拍板**：6 项 → **7 项**（加首页 + 加专属服务，部分改名）
   - 加 **Pill 滑动导航动效**（弹性追随鼠标）
