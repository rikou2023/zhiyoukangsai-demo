### 交互
- 当前页对应链接默认显示电蓝胶囊 Pill
- 鼠标 hover 时 Pill 弹性追随
- 鼠标离开导航 Pill 弹回当前页位置
- 导航容器**透明背景**，只有 Pill（链接背景）有底色

### 右侧 CTA
- **联系我们**：实色 `--bg-elevated` 背景 + 白字 + 描边
- **预约诊断**：白底 + 电蓝字 + 白色发光阴影

## 2. 表单（全站统一）

⚠️ **3 个入口共用同一个表单：**
- 顶栏「预约诊断」按钮
- Hero 区「预约诊断」按钮
- 「联系我们」按钮（顶栏 + 各页 CTA）
- 「需求对接 / 需求收集」页面

### 文案
```
主标题：被 AI 看见，才能被客户选择
副标题：提交您的品牌信息，GEO 专家将为您进行 1V1 深度扫描，
       评估您的品牌在主流 AI 中的推荐指数。
提交按钮：预约专家诊断
```

### 表单字段（全部必填）

| 字段 | 类型 | 说明 |
|---|---|---|
| 公司名称 | text | |
| 姓名 | text | |
| 职位 | text | |
| 联系电话 | tel | |
| 邮箱 | email | |
| 核心需求 | textarea | |
| 目前最关注的 AI 大模型 | **多选** checkbox | Kimi / 文心一言 / 通义千问 / 豆包 / 智谱清言 / DeepSeek |
| 目前品牌在 AI 时代的痛点 | **单选** radio 或下拉 | A/B/C/D 四选一（见下） |

### 痛点 4 选项（一字不改）

- **A** AI 搜不到我们公司/产品
- **B** AI 总是优先推荐竞争对手
- **C** AI 对我们的回答有错误/负面信息
- **D** 想建立专属的企业 AI 知识库

### 提交成功反馈
> 您的需求已收到，会在 3 个工作日内给您反馈。

### 后端
- POST `/api/contact`（已存在）
- Python `server/server.py` 接收，写 `database/leads.xlsx`

## 4. 配色（已锁定，全站禁止使用其它色）

| 变量 | 色值 | 用途 |
